Auto Minesweeper

簡單的自動踩地雷（Minesweeper）示範程式。

執行方式：

```bash
python auto_minesweeper.py
```

說明：
- 檔案 `auto_minesweeper.py` 包含遊戲引擎與自動求解器（簡單邏輯 + 隨機猜測）。
- 可修改遊戲大小與地雷數量：在 `if __name__ == '__main__'` 區塊中調整 `w, h, mines`。
